## 🎯 Problem It Solves

Bad data costs companies $12 million per year.
DataSentinel detects issues before they reach production — automatically.

---

## 💰 Benefits

- Saves hours of manual data checking every week
- Catches data issues before they break ML models
- Reduces data validation effort by up to 60%
- Works on any dataset — HR, sales, finance, healthcare
- Generates professional reports in one click
- Keeps entire data team informed via email alerts
- No technical knowledge needed to use the dashboard

## ✨ Features
| Feature | Description |
|---|---|
| 🔴 Missing Values | Detects empty cells with % per column |
| 🔵 Duplicates | Finds repeated rows instantly |
| 🟡 Outliers | Flags abnormal values automatically |
| 📋 Schema Validator | Checks data types and unique values |
| 🏥 Health Score | Data quality score out of 100 |
| 📄 PDF Report | Downloadable professional report |
| 📧 Email Alerts | Auto alert when quality drops below threshold |

## 🚀 Run Locally
```bash
git clone https://github.com/purva-code24/DataSentinel.git
cd DataSentinel
pip install -r requirements.txt
streamlit run app.py
```

## 📁 Project Structure

- app.py — Streamlit dashboard
- scanner.py — Detection engine
- report.py — PDF generator
- alerts.py — Email alerts
- requirements.txt — Dependencies
- sample_data/ — Sample dataset
