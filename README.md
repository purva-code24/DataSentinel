---

## ✨ Features
| Feature | Description |
|---|---|
| 🔴 Missing Values | Detects empty cells with % per column |
| 🔵 Duplicates | Finds repeated rows instantly |
| 🟡 Outliers | Flags abnormal values automatically |
| 📋 Schema Validator | Checks data types and unique values |
| 🏥 Health Score | Data quality score out of 100 |
| 📄 PDF Report | Downloadable professional report |
| 📧 Email Alerts | Auto alert when quality drops below threshold |

## 🚀 Run Locally
```bash
git clone https://github.com/purva-code24/DataSentinel.git
cd DataSentinel
pip install -r requirements.txt
streamlit run app.py
```
